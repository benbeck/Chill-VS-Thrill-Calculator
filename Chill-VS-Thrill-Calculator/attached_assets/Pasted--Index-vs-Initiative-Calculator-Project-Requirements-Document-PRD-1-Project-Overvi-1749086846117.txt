# Index vs. Initiative Calculator

## Project Requirements Document (PRD)

---

## 1. Project Overview

**Project Name:**  
Index vs. Initiative Calculator

**Purpose & Problem Statement:**  
Entrepreneurs and side hustlers face a classic dilemma: invest in the S&P 500 (the “no-effort” champ) or pour resources into a business venture (the “high-risk, high-reward” challenger). This calculator provides a side-by-side, data-driven showdown—factoring in money, time, and mental effort—to help users make the smartest move.

**Target Users:**  
- Entrepreneurs  
- Side hustlers  
- Anyone deciding between business and stock market investments

**Success Metrics:**  
- Number of calculations run  
- User feedback  
- Time spent on page  
- Shares/embeds

---

## 2. Feature Requirements

### Must-Have Features (MVP)
- Adjustable average annual return for S&P 500 (including reinvested dividends)
- Input for total investment amount
- Input for anticipated annual business profit
- Input for business time commitment (hours/week or total hours)
- Input for cost per hour of your time (to calculate “true” business profit)
- Input for mental effort/resources (simple scale: Low/Medium/High)
- Adjustable investment duration (number of years)
- Calculate button (with default values pre-filled)
- Results display:
  - Side-by-side comparison (S&P 500 vs. business)
  - Bar chart and table updating live with results

### Should-Have Features
- Summary/recommendation (“Based on your inputs, S&P 500 wins!”)
- Option to factor in taxes or fees
- Mobile-friendly/responsive design

### Could-Have Features
- Save/share results (copy link, download image)
- Multiple business scenario comparisons
- More detailed breakdowns (e.g., cash flow per year)

### Won’t-Have Features (Out of Scope)
- Real-time stock data
- User accounts or login
- Complex business modeling (e.g., variable profits, multiple investments)

---

## 3. User Journeys

### Key User Flow
1. User lands on the calculator (no welcome screen—straight to action!)
2. Inputs investment details (defaults provided for quick start)
3. Clicks “Calculate”
4. Sees results instantly:
   - Bar chart and table update
   - Clear, bold summary of which option “wins”
   - Optionally, a fun animation or highlight

### Screens/Pages
- Single calculator page (all action happens here!)

### Interactions
- Input fields (with default values)
- Calculate button
- Results area (bar chart, table, summary)

---

## 4. Technical Requirements

### Frontend Requirements
- HTML, CSS, JavaScript (for easy embedding and beginner-friendliness)
- Responsive/mobile-friendly design
- Charting library (e.g., Chart.js or Google Charts) for bar charts

### Backend Requirements
- None (all calculations are client-side)

### Database Requirements
- None (no data storage needed)

### Integration Requirements
- None (no external APIs or live data)

### Authentication & Security
- None (no user accounts or sensitive data)

---

## 5. Technical Stack

### Recommended Technologies
- **Frontend:** HTML, CSS, JavaScript
- **Charting:** Chart.js (easy to use, embeddable, and beginner-friendly)
- **Hosting/Embedding:**  
  - Develop in Replit  
  - Export as HTML/JS/CSS  
  - Embed in WordPress using the HTML block

### Rationale for Choices
- HTML/CSS/JS: Simple, universal, and perfect for embedding in WordPress. Great for learning!
- Chart.js: Super beginner-friendly, lots of tutorials, and makes beautiful charts with little code.
- No backend/database: Keeps things simple and fast.
